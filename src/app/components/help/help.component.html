Help

Add a Class
  1. Go to the "Classes" tab
  2. On the top right there should be a button that says "Add A Class", click on it
  3. A drop down menu should appear, pick the department your class should be in
  4. After a brief moment, the courses should load in and then you can choose one
  5. The course will then show all available sections in whelpch you can click on "Add To Schedule" to add the class
  6. The course section should now be added to your classes
  7. Click on the "Classes" tab once again to verify that the class was added

Remove A Class
  1. Go to the "Classes" tab
  2. On the top right there should be a button that says "Manage Classes", click on it
  3. Each class on the top left corner should now have an "x" on it
  4. If you wish to remove on the class simply click on the "x"
  5. A confirmation screen should pop up
  6. Click yes to remove the class

Joining A Session
  1. Go to the "Classes" tab
  2. If the class has an active live session there should be a "Join Session" section on the class, click on it
  3. You should now be redirected to a page where you can see what the captionist is typing

View Recent Transcripts
  1. Go to the "Classes" tab
  2. On each class there should be a "Recent Transcripts" section, click on it
  3. If there are transcipts for the class, the five most recent transcripts should now be displayed underneath the class
  4. Click on the "Recent Transcripts" section once again to helpde the transcripts once again

View All Transcripts
  1. Go to the "Transcripts" tab
  2. The classes you have added should be displayed here
  3. If the class has transcripts it should display them all

Editing Trasncripts
  1. View your transcripts
  2. Click on the transcript you wish to edit
  3. You will now be able to see the transcipt and freely edit as you please
  4. Once you are done, click back to the "Classes" tab
  5. Your recent transcripts should now have a copy next to the original transcript with your edits

Downloading Trasncripts
  1. View your transcripts
  2. Click on the download sign next to the transcript you wish to download
  3. The document should now be downloading into your computer
  4. Open the downloaded document to see the transcript

Deleting Trasncripts
  1. View your transcripts
  2. Click on the trash bin sign next to the transcript you wish to delete
  3. A confirmation screen should pop up
  4. Click yes to delete the transcript
